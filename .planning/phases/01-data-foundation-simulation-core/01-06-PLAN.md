<task type="auto">
  <name>Task 1: Create SABR Lahman database build script</name>
  <files>scripts/build_lahman_db.py</files>
  <action>
Create Python script that:

1. Downloads SABR Lahman database ZIP from sabr.org
   - URL: https://sabr.org/lahman-database (find download link for latest CSV package)
   - Use urllib.request for download (stdlib only, no requests dependency)
   - Show progress during download

2. Extracts CSV files from ZIP
   - Use zipfile module (stdlib)
   - Extract to temp directory

3. Creates SQLite database with required tables
   - People table: playerID, nameFirst, nameLast, bats, throws (and other columns from CSV)
   - Batting table: playerID, yearID, teamID, G, AB, R, H, 2B, 3B, HR, RBI, SB, CS, BB, SO, HBP, SF, SH, GIDP
   - Pitching table: playerID, yearID, teamID, G, GS, W, L, IPouts, H, R, ER, HR, BB, SO, HBP, BFP, WP
   - Teams table: yearID, lgID, teamID, name, BPF, PPF

4. Imports CSV data into tables
   - Use csv.DictReader for reading
   - Use parameterized INSERT for safety
   - Handle empty strings -> NULL conversion
   - Batch inserts for performance (commit every 1000 rows)

5. Creates indexes for common queries
   - Batting: (playerID, yearID), (teamID, yearID)
   - Pitching: (playerID, yearID), (teamID, yearID)
   - People: playerID (PRIMARY KEY)
   - Teams: (teamID, yearID)

Script requirements:
- Use only stdlib (no external dependencies)
- Accept optional --output-path argument (default: data/lahman.sqlite)
- Print progress messages during each step
- Clean up temp files after completion
- Exit with error code if download or parsing fails

The SABR download page may require finding the actual download URL. The script should document the expected URL format and fail gracefully if the URL changes.
