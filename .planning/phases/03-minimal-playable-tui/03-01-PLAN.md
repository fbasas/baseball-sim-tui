    Create BaseballSimApp class inheriting from textual.app.App.

    Key bindings per CONTEXT.md decisions:
    - Space and Enter both map to "advance" action (next at-bat)
    - F key maps to "fast_forward" action (simulate rest of game)
    - Q key maps to "quit" action (exit)

    App structure:
    ```python
    from textual.app import App, ComposeResult
    from textual.widgets import Header, Footer, Static

    class BaseballSimApp(App):
        """Main TUI application for baseball simulation."""

        CSS_PATH = "styles/game.tcss"

        BINDINGS = [
            ("space", "advance", "Next Play"),
            ("enter", "advance", "Next Play"),
            ("f", "fast_forward", "Fast Forward"),
            ("q", "quit", "Quit"),
        ]

        def compose(self) -> ComposeResult:
            yield Header()
            yield Static("Game dashboard loading...", id="placeholder")
            yield Footer()

        def action_advance(self) -> None:
            """Advance game by one at-bat (placeholder)."""
            pass

        def action_fast_forward(self) -> None:
            """Simulate rest of game (placeholder)."""
            pass

    if __name__ == "__main__":
        app = BaseballSimApp()
        app.run()
    ```

    NOTE: The compose() method uses a Static placeholder for now. GameScreen will be pushed in Plan 03 after widgets are ready.
