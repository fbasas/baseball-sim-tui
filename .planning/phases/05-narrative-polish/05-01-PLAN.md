  <action>
1. Create `src/game/lineup_builder.py` with:

   `build_lineup(team: Team, repo: LahmanRepository) -> None`:
   - Call `repo.get_appearances(team.info.team_id, team.info.year)` to get position data
   - **Position assignment (greedy, no duplicates):**
     - Map position columns to Position enum: G_c->CATCHER, G_1b->FIRST_BASE, G_2b->SECOND_BASE, G_3b->THIRD_BASE, G_ss->SHORTSTOP, G_lf->LEFT_FIELD, G_cf->CENTER_FIELD, G_rf->RIGHT_FIELD
     - For each position, find player with most games at that position who: (a) is in team.get_available_batters(), (b) not already assigned to another position
     - If tie, use total at_bats from BattingStats as tiebreaker (more AB = more regular)
     - If a position has no candidates, assign player with most G_all who is unassigned
   - **Batting order heuristic:**
     - Calculate OBP for each starter: (hits + walks) / (at_bats + walks) from BattingStats
     - Calculate SLG: (singles + 2*doubles + 3*triples + 4*home_runs) / at_bats
     - Slot 1 (leadoff): highest OBP among speed positions (CF, SS, 2B)
     - Slot 2: second highest OBP overall
     - Slot 3: highest batting average among remaining
     - Slot 4 (cleanup): highest SLG or HR among remaining
     - Slot 5: next highest SLG among remaining
     - Slots 6-8: remaining sorted by batting average descending
     - Slot 9: weakest hitter (lowest OBP)
   - **Starting pitcher:** `team.get_available_pitchers()` sorted by `games_started` descending; pick first
   - Call `create_lineup(team, batting_order, positions, pitcher_id)` to set `team.lineup`
